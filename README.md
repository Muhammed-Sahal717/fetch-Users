# Fetch Users App

A simple JavaScript project that fetches user data from an API and displays it in the browser.

---

## Features

- Fetches users from a public API
- Shows a loading message while data is being fetched
- Displays user names in a list
- Handles network and HTTP errors safely

---

## Concepts Used

- `fetch` API
- `async / await`
- Promises
- Error handling with `try / catch`
- DOM manipulation
- Safe rendering using `textContent`

---

## Project Structure
```
.
├── index.html
├── script.js
└── README.md
```


---

## How It Works

1. Data is fetched from an API using `fetch`.
2. The response is converted to JSON.
3. User names are rendered dynamically in the DOM.
4. Loading and error states are handled properly.
